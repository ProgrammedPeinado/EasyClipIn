Small local application that allows users to quickly copy and paste named links, by adding them to a list, and once clicked they get copied and are available for pasting at convenience. This application also allows users to save and import existing .txt list files. Finally, the application will stay over other applications for easy of access. 

I made this project as I was submitting job applications and I found the process of gathering every single resource, project, and description repetitive and monotonous. Therefore, I made this to help me submit applications more expeditiously.

Requirements:
You'll need the latest JDK to run the JAR properly. Also, make sure to set the JAVA_PATH to your OS's environment variables.
